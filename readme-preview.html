🐾 Pet Adoption Website

A modern, responsive pet adoption website built with a "vibe coding" approach.

✨ Features

  • 🐶 Browse adoptable pets with filtering options
  • 📱 Fully responsive design for all devices
  • 🎨 Modern, clean UI with a focus on user experience
  • 🚀 Fast loading and performance optimized

🧰 Tech Stack

  • HTML5 - Semantic markup
  • Tailwind CSS - Utility-first CSS framework
  • JavaScript - Vanilla JS for interactivity
  • Google Fonts - Source Sans 3 for typography

🌈 What is Vibe Coding?

Vibe coding is an approach to web development that prioritizes the overall feel and experience of a website over rigid technical specifications. It's about creating a cohesive aesthetic and emotional response through:

  • Intuitive Design: Creating interfaces that feel natural and enjoyable to use
  • Emotional Connection: Using colors, typography, and animations that evoke specific feelings
  • Consistent Aesthetics: Maintaining a unified visual language throughout the application
  • User-Centered Focus: Prioritizing how the user feels when interacting with the site

In this project, vibe coding is expressed through:

  • Soft color palette with vibrant accents
  • Smooth transitions and subtle animations
  • Consistent spacing and typography
  • Thoughtful micro-interactions

🚀 Getting Started

  1. Clone the repository 
    git clone https://github.com/yourusername/pet-adoption.git
cd pet-adoption
  2. Open the project in your favorite code editor
  3. Start a local development server 
    # Using Node.js
npm install
npm start
  4. Open your browser and navigate to http://localhost:3000
